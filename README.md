 # 🎧 Call-Center-Performance-and-Efficiency-Dashboard-Jan-Feb-
 This Power BI project analyzes a call center's performance from January to March. It provides insights into the company's  agent performance and call efficiency, which can be used to make executive decisions. It visualizes trends in volume,      satisfaction, resolution, and agents, offering deep insights into the company's operational patterns.

 ---

 ## 📂 Project Overview
 This dashboard was developed to provide a comprehensive view of call center performance using real operational data. It    covers 4,054 inbound calls and presents insights across three analytical pages designed for operational managers and team  leads. The dashboard answered critical questions concerning the call center's performance and efficiency over 3 months:
  - How have total calls varied over time?
  - How have total calls varied by agent and by topic?
  - What was the resolution rate % by each topic?
  - What has been the customer satisfaction provided per agent?
  - How have calls been resolved per agent?
  - What was the number of calls per day?
  - What has been the call volume over the months?
  - What was the average speed of answer overtime?

 ## ⚒️ Tools Used
  - Power BI Desktop – Dashboard design and modelling. 
  - Excel / CSV - Data ingestion.
  - DAX - Custom measures for aggregation, filtering, and time trends.

 ---

 ## 🧹 Data Preparation Steps
 ☑️ Cleaned missing values and ambiguous perpetrator entries. 
 ☑️ Standardized call topics and agent names. 
 ☑️ Create a time dimension. 
 ☑️ Modelled relationships for efficient filtering. 
 ☑️ Built DAX measures for:
  - Total Calls
  - Answer Rate
  - Average Satisfaction
  - Average Speed of Answer
  - Resolution Rate
  - Customer Satisfaction

 ---

 ## 🔎 Dashboard Insights

 ### 📞 Call Volume
  - January recorded the highest call volume at 1,455 calls, followed by a decline and stabilisation at approximately       1,300 calls in February and March.
  - Daily call volume ranged from 100 to 190 calls per day, suggesting potential seasonality in January.

 ### 📝 Call Topics
 Call topics were distributed across five categories:
  - Streaming
  - Payment Issues
  - Technical Support
  - Administrative Support
  - Contract Issues
 The distribution was relatively balanced, indicating diversified customer concerns rather than a single dominant issue.

 ### 🧑‍💻 Agent Workload
 Workload was evenly distributed across all eight agents, with total calls per agent ranging from 484 (Joe) to 536 (Jim).   Minor variations in satisfaction scores present opportunities for internal coaching.

 ### ⚙️ Efficiency
  - The 100% answer rate reflects strong operational management.
  - The 68-second average speed of answer is slightly above industry standards and may be contributing to the neutral
 
 
 satisfaction score of 3.0/5.
